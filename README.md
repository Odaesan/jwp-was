## 개발 진행 방법
* 책의 챕터별 was 요구사항을 파악
* 요구사항에 대한 구현을 완료한 후 >> 자신의 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 코드 리뷰 요청
* 코드 리뷰 피드백에 대한 개선 작업을 하고 다시 PUSH
* 모든 피드백을 완료하면 다음 단계를 도전하고 앞의 과정을 반복
